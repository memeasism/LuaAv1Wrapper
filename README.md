<h1>
LuaAv0Wrapper
</h1>
<p>
A lua script that is a basic wrapper for ffmpeg and is designed to just be an easy way to encode videos to an open source format.
</p>
<h3>
Current Support
</h3>
<p>
Currently, this script only supports Intel GPU and CPU encoding.
</p>
